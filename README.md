# Aplication CRUD - User

Uma aplicação que faz um CRUD onde é possível deletar informações. Não foi utilizado nenhuma Lib. Aplicação totalmente feita com CSS puro.

## Start BackEnd

```Node.js
    cd backend
    npm install
    npm start
```

## Start Frontend

```Node.js
    cd frontend
    npm install
    npm start
```
